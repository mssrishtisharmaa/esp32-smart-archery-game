# Setup Guide

## Prerequisites

- ESP32 Development Board
- Arduino IDE
- Firebase Project
- Modern Web Browser

## Installation

1. Clone the repository.
2. Configure your Firebase project.
3. Update Firebase credentials in `config.js`.
4. Open `web/index.html` in your browser (or serve it locally).
5. Connect the ESP32 to Wi-Fi and start the game.

## Notes

- Ensure the ESP32 and browser are connected to the same network if required by your implementation.
- Configure Firebase Realtime Database rules appropriately for development and production.
